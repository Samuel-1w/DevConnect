# ⟨ DevConnect /⟩

A developer community platform where developers can create profiles, share posts, and connect with each other.

## 🚀 Features

- **User Registration** — sign up with full validation and instant inline error feedback
- **Live Profile Preview** — see your profile update in real time as you type
- **Developer Profile** — dynamically populated from stored data, no hardcoded values
- **Community Board** — create, like, and delete posts with no page refresh
- **Data Persistence** — all data saved to localStorage and reloads after refresh
- **Responsive Design** — works across desktop, tablet, and mobile screens
- **Hamburger Menu** — mobile-friendly navigation with smooth toggle
- **Scroll Reveal** — cards animate in as you scroll down the page
- **Typing Effect** — hero heading types itself out on page load
- **Post Animation** — button celebrates with colour change on successful post
- **Loading State** — signup button shows loading feedback before redirecting
- **Character Counter** — live 280 character limit on posts
- **jQuery Animations** — fade-in posts, slide toggle sections, nav hover effects

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (Vanilla)
- jQuery (via CDN)
- Browser localStorage
